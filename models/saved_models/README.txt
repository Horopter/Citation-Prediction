This directory contains trained model checkpoints (.pkl files):
- model_xgboost_all_features_best.pkl
- model_lightgbm_all_features_best.pkl
- model_catboost_all_features_best.pkl
- model_mlp_all_features_best.pkl
- model_pytorch_mlp_all_features_best.pkl

Each model file contains:
- Trained model
- Scaler
- PCA models (if applicable)
- Best hyperparameters
- Optimal threshold
- Feature column names

Models are generated by running the training notebooks in src/notebooks/.
